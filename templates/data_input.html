If you want to contribute to the data set, please enter your tested data with a picture of the mask that was tested.

**IMPORTANT**
For any data that doesn't have a result, please put "null" in that category.
For "Single-pass filtration efficiency" and "Error Bar", round the data to the nearest hundredth digit.
For the rest of the data, round it up to the nearest tenth digit.
The first row is an example, you may click on the first row to input the data.

After the data is approved, you can see your data in the Data section.
